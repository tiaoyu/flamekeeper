# 守焰者 / Flame Keeper

> **在永恒的黑暗中，守护人类最后的希望之火。**

## 游戏简介

**守焰者** 是一款融合了 **塔防** 与 **即时建造** 的 2D 策略生存游戏。

世界被永恒的黑暗吞噬，文明崩塌，怪物横行。你是最后的守焰者，守护着人类仅存的火种——唯一能驱散黑暗的光芒。

## 核心玩法

- 🔥 火种照亮周围区域
- 🪵 采集资源（木材、石头、铁矿、水晶）
- 🏗️ 建造防御（城墙、箭塔、投石塔...）
- 👾 抵御黑暗中涌来的怪物
- ⬆️ 升级火种 → 扩大照亮范围 → 解锁新建筑

## 游戏特色

| 特色 | 描述 |
|------|------|
| **光明即领土** | 只有火种照亮的区域才能建造 |
| **四面受敌** | 怪物从地图四周的黑暗中涌来 |
| **资源建造链** | 采集 → 加工 → 建造 |
| **波次挑战** | 每波怪物更强，倒计时带来紧迫感 |
